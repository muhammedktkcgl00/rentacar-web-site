
# Rent A Car Website

Bu proje bir araç kiralama web sitesi şablonudur. Responsive bir tasarıma sahiptir.

## Özellikler

- Anasayfa, Araç listesi, Hakkımızda ve İletişim sayfaları  
- Araç kartları ile kiralanabilir araçların görsel sunumu  
- Mobil ve masaüstü uyumlu yapı  

## Teknolojiler

- HTML  
- CSS
